# museum-thesis-site

אתר Next.js מוכן להעלאה ל־GitHub ול־Vercel.

## העלאה ל־Vercel
1. מעלים את כל הקבצים שבתיקייה הזו ל־GitHub.
2. ב־Vercel בוחרים Import Project.
3. חשוב: Root Directory צריך להיות התיקייה שבה נמצאים `package.json` ו־`app`.
4. Build Command: `npm run build`.
5. Output Directory: להשאיר ריק.

## עריכה
העמוד הראשי נמצא כאן:

`app/page.tsx`

העיצוב הכללי נמצא כאן:

`app/globals.css`
